1

Iniciar o Servidor

Execute o servidor local para servir a galeria ao Chrome:

npm run server

Você verá:

🚀 ImgBiel Servidor rodando em http://localhost:3000
⚠️ Importante: Deixe este terminal aberto enquanto usar a extensão!
2

Abrir Extensões no Chrome

Passo A: Abra o Google Chrome

Passo B: Digite na barra de endereço:

chrome://extensions/

Passo C: No canto superior direito, ative "Modo de Desenvolvedor"

3

Carregar a Extensão

Passo A: Clique em "Carregar extensão sem empacotamento"

Passo B: Navegue até a pasta:

C:\Users\guilh\Desktop\imgBiel\chrome-extension

Passo C: Selecione a pasta e clique "Selecionar Pasta"

✅ Sucesso! A extensão aparecerá no Chrome com o ícone 🖼️
4

Usar a Extensão

Passo A: Clique no ícone 🖼️ da extensão (canto superior direito)

Passo B: Clique em "📱 Abrir Galeria"

Passo C: A galeria abrirá em uma nova aba!

✨ Agora você pode usar todas as funcionalidades do ImgBiel!
❓

Problemas Comuns

❌ Extensão não aparece?

→ Verifique se "Modo de Desenvolvedor" está ATIVADO
→ Tente novamente carregar a extensão

❌ Erro "Servidor não está rodando"?

→ Execute no terminal: npm run server
→ Aguarde a mensagem de sucesso
→ Deixe o terminal aberto

❌ Erro ao conectar?

→ Verifique se a porta 3000 está disponível
→ Feche outros programas usando a porta 3000
→ Reinicie o servidor

❌ Imagens não aparecem?

→ Clique em 📁 "Selecionar Pasta"
→ Escolha a pasta com suas imagens
→ Clique 🔄 "Recarregar"

⚡ Atalho Rápido

Clique duas vezes em:

iniciar-servidor.bat
