🚀 COMO USAR:
════════════════════════════════════════════════════════════════════════════════

OPÇÃO 1: Atalho Rápido (Recomendado!)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Double-click em: iniciar-servidor.bat
2. Abra o Chrome e vá para: chrome://extensions/
3. Ative "Modo de Desenvolvedor"
4. Clique "Carregar extensão sem empacotamento"
5. Selecione pasta: chrome-extension
6. Clique no ícone 🖼️ da extensão
7. Clique "📱 Abrir Galeria"

OPÇÃO 2: Manual (Para desenvolvedores)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Terminal 1:
  npm run server

Terminal 2 (Chrome):
  1. chrome://extensions/
  2. Ative "Modo de Desenvolvedor"
  3. "Carregar extensão sem empacotamento"
  4. Selecione chrome-extension/
  5. Clique no ícone e "Abrir Galeria"

════════════════════════════════════════════════════════════════════════════════
📋 ESTRUTURA DA EXTENSÃO:
════════════════════════════════════════════════════════════════════════════════

manifest.json
└─ Define permissões, ícones e interface

popup.html / popup.js
└─ Interface visual com botões:
   ├─ 📱 Abrir Galeria (abre a galeria em nova aba)
   └─ 🚀 Iniciar Servidor (verifica status do servidor)

background.js
└─ Service Worker que:
   ├─ Monitora o servidor
   ├─ Responde a mensagens
   └─ Gerencia a comunicação

server.js
└─ Servidor Express que:
   ├─ Serve arquivos da pasta files/
   ├─ Rota para Nova pasta/
   ├─ Endpoint /health (verificação)
   └─ CORS habilitado
